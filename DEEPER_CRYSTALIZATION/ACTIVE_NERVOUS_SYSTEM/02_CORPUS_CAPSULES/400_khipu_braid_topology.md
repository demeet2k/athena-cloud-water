## Braid Group B_n

The Artin braid group on n strands is generated by n-1 crossings {sigma_1, ..., sigma_{n-1}} satisfying:

- **Far commutativity**: sigma_i * sigma_j = sigma_j * sigma_i when |i-j| >= 2
- **Yang-Baxter relation**: sigma_i * sigma_{i+1} * sigma_i = sigma_{i+1} * sigma_i * sigma_{i+1}

Unlike the symmetric group S_n (which only tracks endpoints), B_n preserves the full crossing history -- exactly the property needed for a memory system that records process, not just outcome.

## Topological Invariants as Data Channels

- **Linking number**: Pairwise relation between cord pairs (signed crossing count)
- **Writhe**: Total signed crossings of a single cord (polarity/direction data)
- **Jones polynomial**: Complete isotopy invariant distinguishing cord configurations
- **Braid index**: Minimum strand count needed to represent the knot (complexity measure)

These invariants are preserved under physical deformation, making the Khipu a **topologically robust** storage medium -- data survives wear, bending, and aging as long as the crossing topology is maintained.
